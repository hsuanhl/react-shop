# LIVIN Store | 現代簡約家居電商平台

一個電商展示專案。透過 React 與 Vite 構建，實現了從前台流暢購物到後台產品管理的完整流程，並專注於極簡的 UI/UX 互動體驗。

---

## 品牌理念
**「把生活，過成喜歡的樣子。」** LIVIN Store 不僅是家具電商，更是一種生活態度的提案。設計上採用大量的留白、大地色系與優雅的過渡動畫，旨在為使用者營造出平靜、溫暖且具質感的購物氛圍。

---

## 技術重點與實作功能

### 1. 狀態管理與全域通知 (Redux Toolkit)
* **全域通知系統**：利用 RTK 實作同步與非同步的通知機制。當使用者執行加入購物車時，系統會自動 Dispatch 全域 Toast 訊息。
* **購物車狀態同步**：即時計算購物車數量與金額，確保全站導覽列狀態一致。

### 2. 前台購物體驗
* **動態 Banner (Swiper.js)**：首頁採用 Swiper 結合效果，提升品牌視覺張力。
* **產品導購流程**：實作產品列表篩選、單一產品詳細頁 (Dynamic Routing)，並串接 RESTful API 進行即時資料抓取。
* **品牌內容經營**：整合了「關於我們」、「常見問題 (FAQ)」、「生活誌」與「實體店面資訊」等功能頁面，模擬真實電商營運深度。

### 3. 管理後台與安全性
* **身份驗證**：實作 Admin 登入/登出功能，透過 Cookie 儲存 Token 確保管理權限安全性。
* **產品 CRUD 管理**：管理員可於後台進行產品建立、編輯、刪除及圖片上傳，並即時同步至前台頁面。

### 4. 效能與視覺優化 (UI/UX)
* **響應式設計 (RWD)**：全站針對手機、平板、桌機進行適配，確保在各類裝置上皆有極佳的閱讀性。

## 開發工具
* **前端框架**: React.js (Hooks)
* **建構工具**: Vite
* **樣式處理**: Tailwind CSS
* **套件整合**: Axios, React Router, Swiper.js, Redux Toolkit
* **API 來源**: 六角學院 API 伺服器

## 安裝與環境設定

1. **複製本專案**
2. **安裝相關套件**
npm install
3. **環境變數配置**
請於根目錄建立 .env 檔案並填入以下資訊：
程式碼片段
VITE_API_BASE=URL
VITE_API_PATH=你的PATH
4. **運行專案**
npm run dev
